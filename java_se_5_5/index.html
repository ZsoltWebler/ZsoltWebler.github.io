    Java Enum típus

    Bevezetés

    Az ENUM kulcsszó a Java 5-s verziójában került bevezetésre. Ez egy speciális osztálytípust jelöl, amely mindig kiterjeszti a java.lang.Enum osztályt. Az enum típus (magyarul felsorolás típus) egy speciális adattípus, amely lehetővé teszi, hogy egy változó előre meghatározott állandók halmazából kaphasson csak értéket. A változónak meg kell egyeznie a számára előre meghatározott értékek egyikével.

    A gyakori példák az enum-ok használatára például az égtájak eltárolása (NORTH, SOUTH, EAST, WEST) vagy a hét napjai.

    Mivel az értékek konstansok, ezért az enum értékeket nagybetűvel írjuk.

    Enumok használata

    A Java nyelvben az enum kulcsszó használatával határozhatunk meg egy enum típust. Például a hét napjainak enum típusát a következőképpen adhatjuk meg:

    public enum Day {
    SUNDAY, MONDAY, TUESDAY, WEDNESDAY,
    THURSDAY, FRIDAY, SATURDAY
}

    Az enum típust, akkor használjuk, ha a típus amit reprezentálni szeretnénk egy előre meghatározott halmaz értékei. Ilyen lehet például a valóságban is előforduló halmaz például a naprendszerünk bolygói, vagy elvont fogalmak zárt halmaza, például egy rendelés aktuális fázisa.

    Az így definiált konstansok olvashatóbbá teszik a kódot, lehetővé teszik a fordítási időben történő ellenőrzést, előre dokumentálják az elfogadott értékek listáját, és elkerülik az érvénytelen értékek átadása miatti váratlan viselkedést.

    A Java nyelv enum típusai eltérnek a más nyelvekben megszokott enum típusoktól. Mivel minden enum típus a java.lang.Enum osztályból származik, ezért vannak előre definiált metódusaink és adattagjaink, amiket az Enum osztálytól öröklünk. Ilyen örökölt metódus például a statikus values() metódus, ami az enum lehetséges értékeinek listájával tér vissza.

    public class Main {
    public enum Day {
        SUNDAY, MONDAY, TUESDAY, WEDNESDAY,
        THURSDAY, FRIDAY, SATURDAY
    }

    public static void main(String[] args) {

        for (Day day : Day.values()) {
            System.out.println(day);
        }

    }
}
