    Java Enum típus

    Bevezetés

    Az ENUM kulcsszó a Java 5-s verziójában került bevezetésre. Ez egy speciális osztálytípust jelöl, amely mindig kiterjeszti a java.lang.Enum osztályt. Az enum típus (magyarul felsorolás típus) egy speciális adattípus, amely lehetővé teszi, hogy egy változó előre meghatározott állandók halmazából kaphasson csak értéket. A változónak meg kell egyeznie a számára előre meghatározott értékek egyikével.

    A gyakori példák az enum-ok használatára például az égtájak eltárolása (NORTH, SOUTH, EAST, WEST) vagy a hét napjai.

    Mivel az értékek konstansok, ezért az enum értékeket nagybetűvel írjuk.

    Enumok használata

    A Java nyelvben az enum kulcsszó használatával határozhatunk meg egy enum típust. Például a hét napjainak enum típusát a következőképpen adhatjuk meg:

    public enum Day {
    SUNDAY, MONDAY, TUESDAY, WEDNESDAY,
    THURSDAY, FRIDAY, SATURDAY
}

    Az enum típust, akkor használjuk, ha a típus amit reprezentálni szeretnénk egy előre meghatározott halmaz értékei. Ilyen lehet például a valóságban is előforduló halmaz például a naprendszerünk bolygói, vagy elvont fogalmak zárt halmaza, például egy rendelés aktuális fázisa.

    Az így definiált konstansok olvashatóbbá teszik a kódot, lehetővé teszik a fordítási időben történő ellenőrzést, előre dokumentálják az elfogadott értékek listáját, és elkerülik az érvénytelen értékek átadása miatti váratlan viselkedést.

    A Java nyelv enum típusai eltérnek a más nyelvekben megszokott enum típusoktól. Mivel minden enum típus a java.lang.Enum osztályból származik, ezért vannak előre definiált metódusaink és adattagjaink, amiket az Enum osztálytól öröklünk. Ilyen örökölt metódus például a statikus values() metódus, ami az enum lehetséges értékeinek listájával tér vissza.

    public class Main {
    public enum Day {
        SUNDAY, MONDAY, TUESDAY, WEDNESDAY,
        THURSDAY, FRIDAY, SATURDAY
    }

    public static void main(String[] args) {

        for (Day day : Day.values()) {
            System.out.println(day);
        }

    }
}

    Az enum típus értékei lényegében az enum osztályunk objektumai. Az enum osztályunk is rendelkezhet adattagokkal és metódusokkal amiket különböző értékekkel inicializálhatunk az objektumok készítésre során.

    Nézzük a következő példát, amiben a Planet enumot hozzuk létre, ami a naprendszerünk bolygóit fogja tárolni.

    public enum Planet {
    MERCURY(3.303e+23, 2.4397e6),
    VENUS(4.869e+24, 6.0518e6),
    EARTH(5.976e+24, 6.37814e6),
    MARS(6.421e+23, 3.3972e6),
    JUPITER(1.9e+27, 7.1492e7),
    SATURN(5.688e+26, 6.0268e7),
    URANUS(8.686e+25, 2.5559e7),
    NEPTUNE(1.024e+26, 2.4746e7);

    private final double mass;   // in kilograms
    private final double radius; // in meters

    Planet(double mass, double radius) {
        this.mass = mass;
        this.radius = radius;
    }

    private double mass() {
        return mass;
    }

    private double radius() {
        return radius;
    }

    // universal gravitational constant  (m3 kg-1 s-2)
    public static final double G = 6.67300E-11;

    double surfaceGravity() {
        return G * mass / (radius * radius);
    }

    double surfaceWeight(double otherMass) {
        return otherMass * surfaceGravity();
    }
}

    A korábbi példával ellentétben itt az enum típusunkhoz adattagokat és metódusokat is definiálunk. Mivel az enum csak egy különleges osztály ezt minden további gond nélkül megtehetjük. Minden egyes enum értékünk mostmár eltárolja az adott bolygó tömegét és a bolygó sugarát és a két metódusán keresztül, minden bolygón kiszámolhatjuk a felületi gravitációt valamint egy tömeg súlyát az adott bolygó felszínén.

    Készíts egy Card osztályt, a következő adattagokkal. String rank, CardSuit cardSuit. Készíts paraméteres konstruktort, toString és getter-setter metódusokat.

    A main függvényedben generálj ki egy 52 lapos paklit az összes lehetséges kártyalappal és tárold el a lapokat egy tömbben!

    3. Feladat

    Készíts egy PizzaSize enumot a következő értékekkel SMALL, MEDIUM, LARGE, EXTRALARGE.

    Készíts egy Pizza osztályt, a következő adattagokkal. String pizzaName, int basePrice, PizzaSize pizzaSize. Készíts paraméteres konstruktort, toString és getter-setter metódusokat. Készíts egy getPrice metódust, ami a pizza árát számolja ki az alábbi képlet segítségével.

    A pizza ára a basePrice * pizza méretével.

    • 1 - SMALL
    • 1.25 - MEDIUM
    • 1.5 - LARGE
    • 2 - EXTRALARGE

    Vagyis egy LARGE pizza, aminek az alap ára 1000 volt, 1500-ba kerül.
