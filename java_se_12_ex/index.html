    Java Swing gyakorló feladatok

    1. Feladat

    Készíts egy Java Swing alkalmazást, amiben pozitív egész számok faktoriálisát számolod ki. Készíts egy beviteli mezőt, ahol a felhasználó meg tudja adni a számot aminek a faktoriálisát ki szeretné számolni és legyen egy szöveg mező, ahol megjeleníted a szám faktoriálisát. A megvalósítás történhez gomb segítségével vagy a szövegmező eseményfigyelésével is.

    2. Feladat

    Készíts egy Java Swing alkalmazást, ahol kocka dobást kell szimulálnod. A felhasználó megadhatja a kocka típusát és hogy hány darab kockát szeretne dobni. A gomb megnyomására pedig egy előugró üzenetbe jelenítsd meg a kapott számot.

    3. Feladat

    Készíts egy programot, ami különböző autók C02 kibocsájtási adatait olvassa be és dolgozza fel. Az alkalmazás elkészítése során a FuelType enumot és a CarEmissionInfo osztályt használd.

    public enum FuelType {

    REGULAR_GASOLINE, //X
    PREMIUM_GASOLINE, //Z
    DIESEL, // D
    E85, //E
    NATURAL_GAS, // N
    UNKNOWN
}

    public class CarEmissionInfo {

    private final String manufacturer;
    private final String model;
    private final double engineSize;
    private final FuelType fuelType;
    private final int co2Emission;


    public CarEmissionInfo(String manufacturer, String model, double engineSize, FuelType fuelType, int co2Emission) {
        this.manufacturer = manufacturer;
        this.model = model;
        this.engineSize = engineSize;
        this.fuelType = fuelType;
        this.co2Emission = co2Emission;
    }


    //... Getter, setter, toString metódusok
}

    A feladatod az emission.csv fájl beolvasása és a megfelelő objektumot létrehozása a rekordok alapján.

    A feldolgozás során, nem kell az összes oszlopot felhasználni!

    A feldolgozott adatokat jelenítsd meg egy Swing alkalmazásban. Készíts egy JFrame-t, ami egy táblázatot tartalmaz és egy gombot. A táblázatban jelenítsd meg a beolvasott CarEmissionInfo lista elemeit a gomb megnyomására pedig nyíljon meg egy új JDialog. ahol egy új CarEmissionInfo objektum adatait lehet megadni és az Ok gomb megnyomására kerüljön be a táblázatba.
