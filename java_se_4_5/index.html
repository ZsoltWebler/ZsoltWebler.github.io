    Konzol io

    Bevezetés

    Ebben a rövid fejezetben a konzol használatával fogunk megismerkedni. Megtanuljuk, hogyan tudunk felhasználói bemenetet kiolvasni a konzolról és hogy hogyan tudunk a konzolra kiírni az adatokat.

    Beolvasás a konzolról

    A konzolról való beolvasáshoz a Scanner osztályt fogjuk használni, ami a java.util csomagban található.

    Scanner scanner = new Scanner(System.in);

    A beolvasáshoz létre kell hoznunk egy Scanner példányt, aminek a konstuktora valamilyen InputStream objektumot vár. A Scanner osztályt fáljokból történő beolvasáshoz is használhatjuk, azonban egyelőre csak a konzolról való olvasással ismerkedünk meg. Ahhoz, hogy a konzolt elérjük paraméternek a System.in-t adjuk meg, ezzel nyerünk hozzáférért az konzol input streamjéhez.

    String line = scanner.nextLine();

    A konzolról való beolvasás soronként történik, ha írunk valamit a konzolon az enter leütéséig egy sornak számít. Egy teljes sor beolvasásához a scanner.nextLine() metódust kell meghívnunk, ami a teljes sorral tér vissza String ként.

    String word = scanner.next();

    Lehetőségünk van a szavak egymás utáni beolvasására is, ehhez a scanner.next() metódust használjuk, így a beolvasás csak az első elválasztó karakterig fogja beolvasni a az inputunkat.

    int age = scanner.nextInt();
double height = scanner.nextDouble();

    Ha valamilyen numerikus inputra számítunk, akkor a Scanner osztály ebben is segítséget nyújt. Ha egész számot szeretnénk beolvasni használjuk a scanner.nextInt() metódust, lebegőpontos számok esetén pedig a scanner.nextDouble() lesz a segítségünkre.

    while (scanner.hasNextInt()) {
    int nmbr = scanner.nextInt();
    //...
}

    Ha be szeretnénk olvasni valamit, de a konzolon nincs ilyen input, akkor hibát kapunk és a program működése leállhat. A Scanner osztálynak, ezért vannak scanner.hasNext_Prefix() metódusai, amik igazzal térnek vissza, ha a konzolról való beolvasás lehetséges. Az összes beolvasó metódushoz, van egy megfelelő ellenőrző metódus is.

    Kiíratás a konzolra

    A konzolra való kiíratáshoz a már ismert System.out osztályt fogjuk használni. 

    System.out.println("Please enter your name and surname: ");

    A System.out.println metódussal teljes sorokat tudunk kiírni, vagy az üzenetünk végén egy sortörés következik.

    System.out.print("Have a good");
System.out.print(" one!");

    Ezzel szemben a System.out.print metódus nem tesz sortörést az üzenetünk után, azok folyó szövegként követik egymást.

    Gyakorló feladatok

    1. Feladat:

    Keresd meg és próbáld ki az összes beolvasó metódust!

    2. Feladat:

    Olvasd be a neved szavanként majd írd ki sordított sorrendben a konzolra.

    3. Feladat:

    Készíts egy programot, ami beolvas a konzolról számokat, majd kiírja ezeknek az átlagát. A program első körben kérje be a felhasználótól, hogy hány számot szeretne átlagolni. Majd utána a felhasználónak egymás után meg kell adnia az n darab számot. Ezután a program írja ki a számot egészét lebegő pontos alakban.

    4. Feladat:

    Írj egy számológépet. A program elindítása után kérdezd meg milyen műveletet szeretne végezni a felhasználó. (+, -, *, /) Ezután kérj be két lebegőpontos számot és végezd el rajtuk a kért műveletet.

    5. Feladat:

    Írj egy számkitalálós játékot. A program generáljon random egy számot (1-100) között, majd a felhasználónak egész számokat kell tippelni, amíg el nem találja a megadott számot. A program adjon segítséget, hogy a tippelt szám nagyobb vagy kisebb mint a kitalálandó szám.
