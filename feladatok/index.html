    Az alábbi RealEstate osztállyal fogunk dolgozni.

    public class RealEstate {

    private final double area;
    private final int numberOfRooms;


    public RealEstate(double area, int numberOfRooms) {
        this.area = area;
        this.numberOfRooms = numberOfRooms;
    }

    public double getArea() {
        return area;
    }

    public int getNumberOfRooms() {
        return numberOfRooms;
    }

}

    Implementáld a toString metódust az osztályhoz.

    Feladatod egy egyszerűsített Könyvtárkezelő Rendszer létrehozása Java nyelven. A rendszernek támogatnia kell két típusú elemet: Könyveket és DVD-ket. Minden elemnek van egy címe, szerzője (könyvek esetén) vagy rendezője (DVD-k esetén), valamint egy egyedi azonosítószáma. Valósítsd meg a rendszert osztályok és enumok segítségével.

    A feladatod a következő osztályok létrehozása:

    Item: Ez az alaposztály mind a könyvek, mind a DVD-k számára. Az alábbi attribútumokkal kell rendelkeznie:

    • int id (egyedi azonosítószám minden elemhez)
    • String title (cím)
    • Egy ItemType enum, ami a BOOK és a DVD értékeket veheti fel.

    Emellett egy konstruktornak kell inicializálnia ezeket az attribútumokat, és megfelelő gettereket és settereket.

    Book: Ez az osztály öröklődik az Item osztályból. Egy további attribútummal kell rendelkeznie:

    • String author (szerző)

    DVD: Ez az osztály öröklődik az Item osztályból. Egy további attribútummal kell rendelkeznie:

    • String director (rendező)

    Library: Ez az osztály képviseli magát a könyvtárat. A következő metódusokat kell tartalmaznia:

    • void addItem(Item item) - Hozzáad egy elemet a könyvtár gyűjteményéhez.
    • void displayItems() - Megjeleníti az összes elem információját a könyvtárban.
    • Item findItemById(int id) - Megkeres és visszaad egy elemet azonosítója alapján.

    Feladatod egy Online Vásárlási Rendszer tervezése Java nyelven. A rendszernek támogatnia kell különböző típusú termékeket: Elektronikai cikkeket és Ruházatot. Minden terméknek van egy név, ár és egy egyedi azonosítószáma. Ezenkívül minden termék rendelkezhet specifikus tulajdonságokkal.

    A feladatod a következő osztályok és interfészek létrehozása:

    Product: Ez az alaposztály mind az elektronikai cikkek, mind a ruházati termékek számára. Az alábbi attribútumokkal kell rendelkeznie:

    • int id (egyedi azonosítószám minden termékhez)
    • String name (név)
    • double price (ár)

    Electronics: Ez az osztály öröklődik a Product osztályból. Egy további attribútummal kell rendelkeznie:

    • String brand (márka)

    Clothing: Ez az osztály öröklődik a Product osztályból. Két további attribútummal kell rendelkeznie:

    • String size (méret)
    • String material (anyag)

    Review Ez az osztály a következő adattagokat és metódusokat tartalmazza.

    • String review (vélemény)
    • int rating (pontozás), csak 1 és 5 között vehet fel értékeket.

    A Product osztály egészítsd ki az alábbi metódusokkal és attribútumokkal.

    • Review[] reviews: A vélemények tárolására használt tömb.
    • void addReview(String review, int rating) - Hozzáad egy értékelést a termékhez.
    • double getAverageRating() - Kiszámítja és visszaadja az átlagos értékelést az értékelések alapján.
