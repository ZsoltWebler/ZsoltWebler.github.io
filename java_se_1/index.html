    Java alapok

    Bevezetés

    Ebben a feladatsorban a java programozás alapjaival fogsz megismerdezni. Elsajátítjuk a változók kezelését, megismerkedünk a műveletek írásával és a lineáris programozás első lépésivel. A feladatok elvégzéséhez szükséged lesz egy telepített fejlesztő környezetre és egy beállított Java JDK-ra.

    Változók

    Ebben a részben a programozás egyik legalapvetőbb koncepciójával fogunk megismerkedni, a változókkal. Megtanuljuk létrehozni, módosítani és a műveleteink során felhasználni és megjelentíteni őket.

    Adattípusok

    A Java nyelvben más nyelvekkel ellentétben kötelező minden változónknak meghatározni az adattípusát. A rengeget beépített adattípuson túl, amit a Java támogat lehetőségünk van saját adattípusok létrehozására is, de egyelőre a beépített alapvető adattípusokkal ismerkedünk meg.

    Adattípus Méret Alapérték Példa értékek Értéktartomány
    byte 1 byte 0 1, 18 -128 – 127
    short 2 byte 0 -13, 24 -32768 – 32767
    int 4 byte 0 141 -2147483648 – 2147483647
    long 8 byte 0 14155511 -9223372036854775808 – 9223372036854775807
    float 4 byte 0.0 1.4 Tört számok, 6-7 tizedes jegy pontossággal
    double 8 byte 0.0 1.8 Tört számok, 15 tizedes jegy pontossággal
    boolean 1 bit false true, false Igaz vagy Hamis
    char 2 byte '\u0000' 'c', 'a' Egy karakter az ASCII táblából
    String - null "Hello" Szöveg

    Változók létrehozása

    Egy változó létrehozása jellemzően két részből áll, egy deklarációból és egy inicializálásból. A deklaráció létrehoz egy üres változót a megadott névvel, az inicializáció pedig megadja a változó értékét.

    int i=12;

    Egy int típusú változó létrehozása i néven és a váltózó értékének beállítása 12-re.

    double d=6.5;

    Egy double típusú változó létrehozása d néven és a váltózó értékének beállítása 6.5-re.

    String s="Hello World!";

    Egy String típusú változó létrehozása s néven és a váltózó értékének beállítása a Hello World! szövegre.

    Változók értékének módosítása

    Egy változó értékét a létrehozása után módosíthatjuk, de a típusát nem változtathatjuk meg. A változó értékének megváltoztatásához már nem kell az adattípust újból megadni.

    int i=12
i=5

    A korábban létrehozott i változó értékét 12-ről 5-re módosítjuk.

    boolean b=false;
b=true;

    A korábban létrehozott b változó értékét hamisról igazra módosítjuk.

    Operátorok

    Ebben a fejezetben az alapvető operátorok használatát tárgyaljuk.

    Alap operátorok

    Operátor Adattípus Művelet Használat
    = Bármilyen típus Értékadás a = 5
    + Szám/String Összeadás/Összefűzés a = 2 + 3
    - Szám Kivonás a = 5 – 2
    * Szám Szorzás a = 2 * 2
    / Szám Osztás a = 10 / 5
    % Szám Maradékképzés a = 10 % 3
    ! Boolean Negáció b = ! false
    && Boolean Logikai És true && false
    || Boolean Logikai Vagy true || false
    == Bármilyen típus Egyenlőség vizsgálat 5 == 2
    != Bármilyen típus Egyenlőtlenség vizsgálat 5 != 2
    >, >= Szám Nagyobb (egyenlő) 5 > 2
    <, <= Szám Kisebb (egyenlő) 5 < 2

    A == egyenlőség vizsgáló operátort kizárólag primitív típusokkal (pl. int, double, boolean) használjuk. Stringek összehasonlítására a .equals() függvényt használjuk. A == operátor ezeken a típusokon a változó memóriacímét hasonlítja össze, így csak akkor lesz igaz, ha a változót önmagával hasonlítjuk össze.

    String s1=" abcd ";
String s2=" abc ";
boolean egyenloseg=s1.equals(s2);
boolean egyenloseg2=s2.equals(s1);

    A Stringek egyenlőségének vizsgálata a .equals() függvénnyel.

    Gyakorló feladatok

    1. Feladat:

    Minden operátorral végezz legalább két műveletet. Találj ki praktikus, valós problémákat amiket ezekkel az operátorokkal lehet megoldani.

    2. Feladat:

    Nézd meg mi történik, ha:

    • Stringet összefűzöl egy másik, nem String változóval?
    • boolean operátort használsz int változókkal vagy fordítva?
    • két összeadott int eredménye nagyobb mint 2147483647?
    • int és double változókkal vegyesen végzel műveleteket?

    3. Feladat:

    A megfelelő operátorok használatával végezz maradékos osztást. 

    a = 20
b = 4
hanyados = ?
maradek = ?

    4. Feladat:

    Mi lesz a boolean B értéke az alábbi műveletek elvégzése után?

    int a = 4;
int b = 3;
boolean B = a > b;

    int a = 4;
int b = 3;
boolean B = a <= b || a != 3;
