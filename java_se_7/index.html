    Java IO

    Bevezetés

    Ebben a szekcióban megismerkedünk a java.io csomaggal. A java.io a Java programozási nyelv alapvető be- és kimeneti ( I/O) műveletekhez használt csomagja. Az "I/O" a "Input/Output" rövidítése, és az adatok beolvasását (input) és kiírását (output) jelenti az alkalmazások és a külvilág között.

    File osztály

    Az java.io.File osztály a Java I/O csomag része, és lehetővé teszi az alkalmazások számára a fájlok és könyvtárak kezelését a fájlrendszerben. A File osztály nem olvas vagy ír adatokat a fájlokból, hanem a fájlok és könyvtárak manipulálását végzi, például fájl létrehozását, módosítását, törlését stb.

    public class FileExample {
    public static void main(String[] args) {
        try {
            File file = new File("example.txt"); // (1)

            if (file.createNewFile()) { // (2)
                System.out.println("A fájl létrejött.");
            } else {
                System.out.println("A fájl már létezik.");
            }

            if (file.delete()) { // (3)
                System.out.println("A fájl törölve lett.");
            } else {
                System.out.println("A fájlt nem sikerült törölni.");
            }

            File directory = new File("myDirectory"); //(4)
            if (directory.mkdir()) {
                System.out.println("A könyvtár létrejött.");
            } else {
                System.out.println("A könyvtár már létezik vagy nem sikerült létrehozni.");
            }

            System.out.println("Abszolút elérési út: " + file.getAbsolutePath()); // (5)

            System.out.println("Fájl vagy könyvtár: " + (directory.isFile() ? "Fájl" : "Könyvtár")); // (6)


            System.out.println("Létezik: " + file.exists()); // (7)


        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
    1. Fájl vagy könyvtár létrehozása (relatív elérési út) Abszolút elérési út használatára is van lehetőség 
       File file = new File("C:\\Users\\username\\Desktop\\example.txt");
    2. Fájl létrehozása
    3. Fájl törlése
    4. Könyvtár létrehozása
    5. A File objektum abszolút útvonalának elérése.
    6. Ellenőrizzük, hogy egy elem könyvtár-e vagy sem
    7. Ellenőrizzük, hogy az elem létezik-e

    A File osztály további metódusokkal is rendelkezik, amelyek segítenek az elérési utak kezelésében, az elemek ellenőrzésében és a fájl attribútumok lekérdezésében.

    Input és Output

    A Java I/O osztályok adatok beolvasására és adatok kimenetének kezelésére szolgálnak. Ezt a funkciót két alapvető csoportra lehet osztani: az adatok beolvasását és az adatok kimenetét kezelő részekre. Az öröklődés szempontjából az InputStream vagy Reader osztályokból származnak az adatok beolvasásához szükséges osztályok gyűjteménye. Ugyanakkor az OutputStream vagy Writer osztályokból származnak az adatok kimenetét szabályozó eszközök.

    Ezen osztályok közötti hierarchia azt jelenti, hogy az InputStream és Reader osztályok általánosabbak, és az ezekből származó osztályok specializálódnak a különböző típusú bemeneti adatok kezelésére. Hasonlóképpen, az OutputStream és Writer osztályok általánosítják a kimeneti műveleteket, amelyeket az azokból származó osztályok finomítanak a speciális kimeneti formátumokhoz.

    Az adatok beolvasásához és kiírásához az InputStream és OutputStream osztályok a read és write metódusokat biztosítják, amelyek lehetővé teszik byte-ok vagy byte tömbök olvasását és írását. Fontos megérteni, hogy ezek az alapvető osztályok általában nem kerülnek közvetlenül használatra a kódunkban, hanem más, kényelmesebb interfészeket nyújtó osztályokat használunk, például a BufferedReader vagy a BufferedWriter osztályokat.

    Mivel a kényelmes I/O kezelés érdekében gyakran több objektumot kell egymásba ágyazva használnunk, az I/O folyamat megértése időnként bonyolult lehet. Ezért léteznek olyan magasabb szintű osztályok és keretrendszerek, amelyek segíthetnek a hatékonyabb és kényelmesebb I/O műveletek kivitelezésében.

    Beolvasás fájlokból

    import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class FileReadingExample {
    public static void main(String[] args) {
        try {
            File filePath = new File("sample.txt"); // (1)
            FileReader fileReader = new FileReader(filePath); // (2)
            BufferedReader bufferedReader = new BufferedReader(fileReader); // (3)

            String line;

            while ((line = bufferedReader.readLine()) != null) { // (4)
                System.out.println(line);
            }

            bufferedReader.close();// (5)
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
    1. Fájl objektum létrehozása az elérési úttal.
    2. FileReader objektum létrehozása.
    3. BufferedReader objektum létrehozása a FileReader objektum segítségével.
    4. Soronkénti beolvasás és kiíratás
    5. A fájl bezárása

    Ebben a példában a FileReader és a BufferedReader osztályokat használjuk a fájl beolvasásához. Az FileReader osztály a fájl megnyitására és beolvasására szolgál, míg a BufferedReader egy pufferelt karakter olvasót nyújt, amely hatékonyabb olvasást tesz lehetővé.

    A try-catch blokk azért van jelen, hogy kezelje a potenciális I/O kivételeket. Az IOException azokat az eseteket fedi le, amikor hiba történik a fájl olvasása közben.

    A readLine() metódus soronként olvassa a fájlt, és amikor a végére ér, null értékkel tér vissza. Így könnyedén végigolvashatjuk a fájl tartalmát soronként.

    Beolvasás a Scanner osztály használatával

    import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class FileReadingExample {
    public static void main(String[] args) {
        try {
            File filePath = new File("sample.txt"); // (1)
            Scanner scanner = new Scanner(filePath); // (2)

            while (scanner.hasNextLine()) {
                String line = scanner.nextLine(); // (3)
                System.out.println(line); // (4)
            }

            scanner.close();// (5)
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
    1. Fájl objektum létrehozása az elérési úttal.
    2. Scanner objektum létrehozása a File objektum segítségével.
    3. Sor beolvasása a fájlből.
    4. A beolvasott sor kiírása a konzol kimenetre.
    5. A fájl bezárása

    A korábban is használt Scanner osztály a konzolról való beolvasáson kívül lehetővé teszi az egyszerű és kényelmes adatbeolvasást, és különösen hasznos, ha a fájlban egyszerűbb típusú adatok vannak, például szöveges adatok vagy számok.

    Miután a Scanner objektumot létrehoztuk a használata megegyezik azzal, amikor a konzolról való beolvasásra használtuk.

    Írás fájlokba

    import java.io.BufferedWriter;
import java.io.File;
import java.io.FileWriter;
import java.io.IOException;

public class FileWritingExample {
    public static void main(String[] args) {
        try {
            File filePath = new File("output.txt"); // (1)

            FileWriter fileWriter = new FileWriter(filePath); // (2)
            BufferedWriter bufferedWriter = new BufferedWriter(fileWriter); // (3)

            bufferedWriter.write("Ez egy példa arra, hogy hogyan lehet adatot fájlba írni."); // (4)
            bufferedWriter.newLine(); // (5)
            bufferedWriter.write("Még egy sor.");

            bufferedWriter.close(); // (6)

        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
    1. Fájl objektum létrehozása az elérési úttal.
    2. FileWriter objektum létrehozása.
    3. BufferedWriter objektum létrehozása a FileWriter objektum segítségével.
    4. Adat kiírása a fájlba.
    5. Sortörés beszúrása a fájlba.
    6. A fájl bezárása

    Ebben a példában a FileWriter és a BufferedWriter osztályokat használjuk a fájlba írásához. A FileWriter osztály a fájl megnyitására és írására szolgál, míg a BufferedWriter egy pufferelt karakter írót nyújt, amely hatékonyabb írást tesz lehetővé.

    Az IOException-t a try-catch blokkban kezeljük, amely az esetleges I/O hibákat fedi le.

    A write() metódus használható a karakterek vagy karakter sorozatok kiírására a fájlba. A newLine() metódus segítségével új sort illeszthetünk be a fájlba.

    Végül ne felejtsük el a BufferedWriter példányt bezárni, hogy a fájl megfelelően lezáródjon és a változtatások elmentődjenek.

    try-with-resources

    A try with resources újítást az Java 7-ben vezették be, és lehetővé teszi az erőforrások (pl. fájl, hálózati kapcsolat) automatikus bezárását anélkül, hogy kifejezetten le kellene zárnod azokat a finally blokkban.

    import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class TryWithResourcesExample {
    public static void main(String[] args) {
        File filePath = new File("sample.txt"); // (1)

        try (BufferedReader bufferedReader = new BufferedReader(new FileReader(filePath))) { // (2)
            String line;
            while ((line = bufferedReader.readLine()) != null) { // (3)
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
    1. Fájl objektum létrehozása az elérési úttal.
    2. BufferedReader objektum létrehozása a FileReader objektum segítségével a try szerkezeten belül.
    3. Soronkénti beolvasás és kiíratás

    Ebben a példában a BufferedReader példányt a try with resources szerkezetbe helyezzük. A Java biztosítja, hogy a BufferedReader példány automatikusan bezáródik a try blokk végén, függetlenül attól, hogy a blokk végrehajtása során a kivétel keletkezik-e vagy sem.

    Ez a szerkezet jelentősen megkönnyíti az erőforrások kezelését és zárását, és csökkenti az olyan hibák kockázatát, amik a bezárás elmulasztásából származnának.

    CSV fájlok kezelése

    A CSV (Comma-Separated Values) egy olyan szöveges fájlformátum, amelyet gyakran használnak adattáblák, adatlisták vagy táblázatok tárolására és megosztására. A CSV formátum egyszerű, könnyen olvasható és írható, és általában támogatott sokféle alkalmazásban, például szövegszerkesztőkben, táblázatkezelőkben és adatbázis-kezelő rendszerekben.

    A CSV fájl adatai szövegként vannak tárolva, és a cellákat, oszlopokat vagy mezőket a vessző karakterrel (,) választják el egymástól. A CSV fájlban általában minden sor egy rekordot vagy elemet tartalmaz, és a sorokban lévő mezők adatai az adott sorban szintén vesszőkkel vannak elválasztva.

    Vegyük az alábbi csv fájlt.

    sample.csv
    Név,Kor,Ország
Alice,28,USA
Bob,32,UK
Charlie,24,Canada

    Ebben a példában a CSV fájl három oszlopot tartalmaz: Név, Kor és Ország. Minden sor egy egyedi rekordot képvisel, és a mezők a vessző karakterrel vannak elválasztva.

    CSV fájlok feldolgozása

    import java.io.BufferedReader;
import java.io.File;
import java.io.FileReader;
import java.io.IOException;

public class CSVFileProcessing {
    public static void main(String[] args) {
        File filePath = new File("sample.csv"); // (1)

        try (BufferedReader bufferedReader = new BufferedReader(new FileReader(filePath))) { // (2)
            String line;
            while ((line = bufferedReader.readLine()) != null) { // (3)
                String[] fields = line.split(","); // (4)

                System.out.println("Név: " + fields[0] + ", Kor: " + fields[1] + ", Ország: " + fields[2]); // (5)
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
    1. Fájl objektum létrehozása az elérési úttal.
    2. BufferedReader objektum létrehozása a FileReader objektum segítségével a try szerkezeten belül.
    3. Soronkénti beolvasás
    4. Felbontjuk a sort a vessző karakterek mentén.
    5. Kiírjuk az adatokat a mezők szerint

    A korábbi sample.csv fájl beolvasásánál feltételezzük, hogy minden oszlop három értéket tartalmaz. A split() metódus segítségével szétválasztjuk a sort a vessző karakterek mentén, és a kapott mezőket egy tömbben tároljuk.

    CSV fájlok létrehozása

    import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class CreateCSVFile {
    public static void main(String[] args) {
        File filePath = new File("sample.csv"); // (1)

        try (BufferedWriter bufferedWriter = new BufferedWriter(new FileWriter(filePath))) { // (2)

            String header = "Név,Kor,Ország"; // (3)
            bufferedWriter.write(header);
            bufferedWriter.newLine();


            String[] names = {"Alice", "Bob", "Charlie"}; // (4)
            int[] ages = {28, 32, 24};
            String[] countries = {"USA", "UK", "Canada"};

            for (int i = 0; i < names.length; i++) { // (5)
                String line = names[i] + "," + ages[i] + "," + countries[i];
                bufferedWriter.write(line);
                bufferedWriter.newLine();
            }

            System.out.println("CSV fájl sikeresen létrehozva és írva.");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
    1. Fájl objektum létrehozása az elérési úttal.
    2. BufferedWriter objektum létrehozása a FileWriter objektum segítségével a try szerkezeten belül.
    3. Fejléc sor létrehozása és beírása a fájlba.
    4. Adat rekordok létrehozása.
    5. Egy for ciklussal végig megyünk az adat rekordokon és beírjuk őket a csv fájlba.

    Először a fejléc sort hozzáadjuk, majd az adatokat (név, kor, ország) írjuk be a fájlba. A BufferedWriter osztályt használjuk, hogy hatékonyan írjunk a fájlba.
