    String and Date

    Bevezetés

    A Java programozási nyelv egyik legfontosabb és leggyakrabban használt adattípusa a String. A String az egyszerű karakterláncokat, szavakat, mondatokat és akár hosszabb szövegeket is képes reprezentálni. Ebben a fejezetben áttekintjük a Java String osztályának alapvető jellemzőit, műveleteit, és bemutatjuk, hogy hogyan lehet hatékonyan használni a szövegek kezelésére.

    String osztály

    A String osztály egy kiemelten fontos adattípus a Java-ban, és a Java Standard könyvtár része. Az osztály immutábilis, vagyis egyszer létrehozott String objektumot nem lehet megváltoztatni. Ez azt jelenti, hogy a String objektumok értéke nem módosítható, csak új String objektumok hozhatók létre.

    Stringet több féle módon tudunk létrehozni:

            String szoveg1="Hello, World!"; // (1)
        String szoveg2=new String();     // (2)
        String szoveg3=new String("Java"); // (3)
    1. Értékadással történő inicializáció
    2. Üres sztring létrehozása konstuktor hívással
    3. Létrehozás literálból

    Mivel a String az egyik leggyakrabban használt osztály, így a nyelv az automatikus konstruktorhívás lehetőségét is felkínálja, azaz new hívás nélkül, egyszerű értékadással is inicializálható a String egy karaktersorozattal, vagy sztring konstanssal.

    Azonban a Stringek annyival többek már így is a karakter sorozatoknál, hogy értelmezve van rajtuk a "+" operátor, aminek hatására két sztring összefűzhető, és eredményül keletkezik egy harmadik String. Ez a "+" operátor igen érdekes olyan értelemben, hogy a Java a String kivételével nem teszi lehetővé ennek az operátornak a felüldefiniálását.

    String osztály metódusai

    A String osztály számos beépített metódust kínál a szövegek manipulálásához:

    length() - Ez a metódus visszaadja a String hosszát, azaz a benne található karakterek számát.

        String szoveg="Hello, World!";
        int hossz=szoveg.length(); // (1)
    1. A változó értéke 12

    charAt(index) - Ez a metódus visszaadja a megadott indexen található karaktert a String-ből. Az index 0-tól indul.

            String szoveg="Hello, World!";
        char karakter=szoveg.charAt(4); // (1)
    1. A karakter értéke 'o'

    substring(start, end) - Ez a metódus visszaad egy részszöveget a String-ből, a megadott kezdeti és záró index között. A kezdeti index beletartozik a részszövegbe, a záró index viszont már nem.

            String szoveg="Hello, World!";
        String reszszoveg=szoveg.substring(0,5); // (1)
    1. A változó értéke "Hello"

    toUpperCase() és toLowerCase() - Ezek a metódusok a szöveget átalakítják nagybetűssé vagy kisbetűssé

            String szoveg="Hello, World!";
        String nagybetus=szoveg.toUpperCase(); // (1)
        String kisbetus=szoveg.toLowerCase(); // (2)
    1. A változó értéke "HELLO, WORLD!"
    2. A változó értéke "hello, world!"

    equals(other) - Ez a metódus összehasonlítja két String-et tartalmilag. Visszaadja true-t, ha a két szöveg azonos, egyébként false-t.

            String szoveg1="Hello, World!";
        String szoveg2="Hello, World!";
        boolean azonos=szoveg1.equals(szoveg2); // (1)
    1. A változó értéke true

    indexOf(str) - Ez a metódus megtalálja az első előfordulást a megadott részszövegnek a String-ben, és visszaadja az első előfordulás indexét. Ha nem találja meg, -1-et ad vissza.

            String szoveg="Hello, World!";
        int index=szoveg.indexOf("World"); // (1)
    1. A változó értéke 7

    replace(oldStr, newStr) - Ez a metódus kicseréli az összes előfordulást a régi szövegből az új szövegre a String-ben.

            String szoveg="Hello, World!";
        String ujSzoveg=szoveg.replace("World","Java"); // (1)
    1. A változó értéke "Hello, Java!"

    split(delimiter) - Ez a metódus feldarabolja a String-et egy megadott elválasztó (delimiter) alapján, és egy tömbben tárolja az eredményeket.

            String szoveg="Alma,Körte,Szilva";
        String[]gyumolcsok=szoveg.split(","); // (1)
    1. A tömb tartalma: ["Alma", "Körte", "Szilva"]

    trim() - Ez a metódus eltávolítja a String elejéről és végéről az extra szóközöket (és más vezérlő karaktereket). Ez hasznos, ha a szöveget tisztítani szeretnénk a felesleges szóközöktől.

            String szoveg="   Java is awesome   ";
        String trimmed=szoveg.trim(); // (1)
    1. A változó értéke "Java is awesome"

    startsWith(prefix) és endsWith(suffix) - Ezek a metódusok ellenőrzik, hogy a String a megadott prefixszel kezdődik vagy a megadott suffixszel végződik.

            String szoveg="Hello, World!";
        boolean starts=szoveg.startsWith("Hello"); // (1)
        boolean ends=szoveg.endsWith("Java"); // (2)
    1. A változó értéke true
    2. A változó értéke false

    isEmpty() - Ez a metódus ellenőrzi, hogy a String üres-e (nincsenek benne karakterek). Visszaadja a true értéket, ha a String üres, egyébként false-t.

            String uresSzoveg="";
        boolean ures=uresSzoveg.isEmpty(); // (1)
    1. A változó értéke true

    isBlank() - Ez a metódus ellenőrzi, hogy a String üres vagy csak vezérlő karakterekből áll. Visszaadja a true értéket, ha a String üres, egyébként false-t.

            String uresSzoveg="  ";
        String uresTabulator="\t";
        boolean ures=uresSzoveg.isBlank(); // (1)
        boolean uresTab=uresTabulator.isBlank(); // (2)
    1. A változó értéke true
    2. A változó értéke true

    valueOf(data) - Ez a metódus egy String objektumot hoz létre az adat más típusából (pl. int, double) a String reprezentációval.

            int szam=42;
        String szoveg=String.valueOf(szam); // (1)
    1. A változó értéke "42"
