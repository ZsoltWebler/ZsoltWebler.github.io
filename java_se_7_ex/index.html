    2. Feladat

    Készítsünk programot ami a konzolról beolvas szavakat, ad nekik egy sorszámot, majd kiírja a szavak.txt nevű fájlba. Ha a begépelt szó exit akkor kilép. Tehát, ha az első beolvasott szó az volt, hogy macska, akkor a szavak.txt fájlba (ha nem létezik, hozzunk létre egyet kódból) beírja, hogy 1 macska. Ha a következő szó az hogy kutya, akkor a fájlba bekerül, hogy 2 kutya és így tovább, amíg az exit parancs le nem állítja a programot.

    3. Feladat

    Vegyük az alábbi Termek osztályt. Készíts egy statikus writeTermekToCSV() metódust, ami egy termek tömböt kap paraméterül és kiírja azt a termek.csv fájlba. A main() metódusban hozz létre egy tömböt 10 darab termékkel és hívd meg a writeTermekToCSV() metódust.

    class Termek {
    private static int kovetkezoAzonosito = 100000;
    private int azonosito;
    private String megnevezes;
    private int ar;

    public Termek(String megnevezes, int ar) {
        this.azonosito = kovetkezoAzonosito++;
        this.megnevezes = megnevezes;
        this.ar = ar;
    }

    public int getAzonosito() {
        return azonosito;
    }

    public String getMegnevezes() {
        return megnevezes;
    }

    public int getAr() {
        return ar;
    }
}

    A Student osztály egy diákot reprezentál. Minden diáknak van egy int id egy String name és egy Subject[] grades adattagja. A feladatod a hiányzó getAverageScore(), getPerformance() és toString() metódusok implementálása.

    • getAverageScore() - Térjen vissza a tantárgyakon elért átlagos pontszámmal.
    • getPerformance() - Hozz létre egy Performance enumot, az alábbi 4 lehetséges értékkel és térj vissza a megfelelő enummal.
      • EXCELENT - Ha az átlag pontszám 90 pont felett van.
      • GOOD - Ha az átlag pontszám 80 pont felett van.
      • AVERAGE - Ha az átlag pontszám 70 pont felett van.
      • BELOW_AVERAGE - Ha az átlag pontszám 70 pont alatt van.
    • toString() - Jelenítse meg a diák adatait a következő formában: 
      Student: Alice
Average Score: 85.0
Performance: Good
